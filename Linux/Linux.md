### 03. 디렉터리와 파일 관리(120p)

>   * [그림3-4]
>   * 리눅스에서 제공되는 디렉터리의 구조는 계층적 형태인 트리로 구성
>   * /"ROOT" : 가장 최상위권에 위치. 가장 조상님

>
>* 작업 디렉터리(122p)
>   * 사용자가 리눅스 시스템에 접속하여 현재 사용하고 있는 디렉터리를 의미
>   * 현재 작업중인 디렉터리는 **점(.)** \으로 표시

>* 홈 디렉터리(123p)

>   * ex) guest_01의 홈 디렉터리 : 'guest_01~'
>
#### 2. 절대 경로와 상대 경로(123p)
>* 절대 경로
>   * 루트(root) 디렉터리인 /를 기준으로 제공되는 디렉터리
>   * /usr/share/zoneinfo/Asia
>   * **경로의 맨 앞에 /가 있으면 절대 경로**
>   * 절대 경로는 항상 루트(/) 디렉터리 위치부터 시작함.
>   
>* 상대 경로
>   * 절대 경로인 루트(/)를 포함하지 않는 경로를 표현
>   * 현재 자신이 속해 있는 디렉터리는 '.'으로 표현
>   * 바로 상위 디렉터리로 올라갈 땐 '..'으로 표현
>   * ex) ../usr/lib
>   * 경로의 맨 앞에 /가 없으면 상대 경로
>   * 사용자가 어느 디렉터리에서 작업하고 있느냐에 따라 경로명이 달라진다.
>   * 상대 경로는 현재 디렉터리를 기준으로 경로명을 시작함.

### 05. 파일 명령어(p148)
#### 1. 파일 생성
>* mkdir [디렉터리명] : 실습 디렉터리 생성
>* touch : 빈 파일 생성
>
#### 2. 파일 복사(154p)
>* cp : copy의 약어. 파일이나 디렉터리 복사할 때 사용
>
>* 원본 파일을 다른 파일 이름으로 복사(155p)
>   * cp/etc/avahi/hosts text_01  : 원본 파일 host를 text_01 파일로 복사
>   
>* 대상 파일을 디렉터리로 복사(157p)
>   1) mkdir temp   : 현재 디렉터리에 새로운 temp 디렉터리 생성
>   2) ls           : temp 디렉터리 확인
>   3) cp text_01 temp  : text_01 파일을 temp 디렉터리에 복사
>   4) ls temp          : temp 디렉터리에 복사한 파일이 존재하는 지 확인
>
>* 대상 파일을 특정 디렉터리로 복사(158p)
>   1) cp text_01 temp/text_02    : 현재 디렉터리에 존재하는 text_01 파일을 temp 디렉터리에 text_02 파일 이름으로 복사
>   2) ls temp                    : ls 명령으로 temp 디렉터리에 복사된 text_02 파일 확인
>
>* [도전문제 3-11] 162p

>      ls
