# 01. 안드로이드의 개요
>## 1. 스마트폰의 개요(20p)

>* 스마트폰
>   * 통화기능 + 컴퓨터 + 다양한 기능 내장(MP3, 카메라, DMB, GPS 등)
>
>* 스마트폰의 역사
>   * 1992년 IBM사의 사이먼(최초)
>   * 1996년 노키아9000
>   * 2002년 마이크로소프트 포켓PG
>   * 2007년 아이폰
>   * 2008년 안드로이드 폰
>   * 2010년 윈도폰7
>
>* 스마트폰의 운영체제(21p)
>   * 2002년 2분기 기준 세계시장 점유율
>   * 안드로이드 74%, 아이폰 25%,  윈도폰 0.01%
>
